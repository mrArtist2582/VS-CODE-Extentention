# Project Structure

Complete overview of the extension's file organization.

```
error-aavi/
│
├── .vscode/                          # VS Code workspace settings
│   ├── extensions.json               # Recommended extensions
│   ├── launch.json                   # Debug configurations
│   └── tasks.json                    # Build tasks
│
├── media/                            # Media assets
│   ├── error-aavi.mp3               # Sound file (YOU MUST ADD THIS)
│   ├── icon.png                     # Extension icon (128x128, optional)
│   └── README.md                    # Media folder documentation
│
├── src/                             # Source code
│   └── extension.ts                 # Main extension logic
│
├── out/                             # Compiled JavaScript (auto-generated)
│   └── extension.js                 # Compiled from src/extension.ts
│
├── node_modules/                    # Dependencies (auto-generated)
│   └── ...                          # Installed via npm install
│
├── .editorconfig                    # Editor configuration
├── .eslintrc.json                   # ESLint rules
├── .gitignore                       # Git ignore rules
├── .npmrc                           # npm configuration
├── .vscodeignore                    # Files excluded from VSIX
│
├── CHANGELOG.md                     # Version history
├── LICENSE                          # MIT License
├── MARKETPLACE.md                   # Publishing guide
├── PROJECT_STRUCTURE.md             # This file
├── QUICKSTART.md                    # Quick start guide
├── README.md                        # Main documentation
├── SETUP.md                         # Detailed setup guide
│
├── package.json                     # Extension manifest & dependencies
├── package-lock.json                # Locked dependency versions
├── tsconfig.json                    # TypeScript configuration
│
└── error-aavi-1.0.0.vsix           # Packaged extension (generated)
```

## 📁 Key Directories

### `/src`
- Contains TypeScript source code
- `extension.ts` - Main entry point with activate/deactivate functions

### `/out`
- Auto-generated by TypeScript compiler
- Contains compiled JavaScript
- Not committed to git (in .gitignore)

### `/media`
- Static assets (sound, icon)
- **IMPORTANT**: Add `error-aavi.mp3` here before building

### `/.vscode`
- VS Code workspace configuration
- Debug and build task definitions

### `/node_modules`
- npm dependencies
- Auto-generated by `npm install`
- Not committed to git (in .gitignore)

## 📄 Key Files

### `package.json`
- Extension manifest
- Defines name, version, publisher
- Lists dependencies
- Configures activation events
- Defines commands and contributions

### `tsconfig.json`
- TypeScript compiler options
- Output directory: `out/`
- Target: ES2020
- Module: CommonJS

### `.vscodeignore`
- Files excluded from VSIX package
- Reduces extension size
- Excludes source files, tests, config

### `extension.ts`
- Main extension code
- `activate()` - Called when extension starts
- `deactivate()` - Called when extension stops
- Terminal monitoring logic
- Sound playback functions

## 🔄 Build Process

```
src/extension.ts  →  [TypeScript Compiler]  →  out/extension.js
                                                      ↓
                                              [VSCE Package]
                                                      ↓
                                          error-aavi-1.0.0.vsix
```

## 📦 VSIX Contents

When packaged, the VSIX includes:
- ✅ `out/` - Compiled JavaScript
- ✅ `media/` - Sound and icon files
- ✅ `package.json` - Extension manifest
- ✅ `README.md` - Documentation
- ✅ `CHANGELOG.md` - Version history
- ✅ `LICENSE` - License file
- ❌ `src/` - Source code (excluded)
- ❌ `node_modules/` - Dependencies (excluded, bundled separately)
- ❌ `.vscode/` - Dev settings (excluded)
- ❌ Config files - tsconfig, eslint, etc. (excluded)

## 🎯 Important Files to Modify

Before publishing, update these files:

1. **`package.json`**
   - `publisher` - Your publisher ID
   - `repository` - Your GitHub URL
   - `version` - Semantic version

2. **`README.md`**
   - Update repository links
   - Add screenshots/GIFs
   - Update contact info

3. **`LICENSE`**
   - Add your name/organization

4. **`media/error-aavi.mp3`**
   - Add your custom sound file

5. **`media/icon.png`** (optional)
   - Add 128x128 extension icon

## 🚫 Files Not to Commit

Already in `.gitignore`:
- `out/` - Compiled code
- `node_modules/` - Dependencies
- `*.vsix` - Packaged extensions
- `.vscode-test/` - Test artifacts

## 📊 File Sizes

Typical sizes:
- `src/extension.ts` - ~7 KB
- `out/extension.js` - ~10 KB
- `media/error-aavi.mp3` - < 500 KB (recommended)
- `media/icon.png` - ~10-50 KB
- `node_modules/` - ~50-100 MB (not in VSIX)
- Final VSIX - ~500 KB - 1 MB
