## ✨ Features

- **🎵 Instant Audio Alerts** - Custom sound notification when terminal errors occur
- **🎯 Smart Detection** - Automatically detects error keywords: `error`, `failed`, `exception`
- **⚡ Zero Configuration** - Works out of the box, no setup required
- **🔧 Test Command** - Built-in test to verify your sound alerts
- **🚀 Lightweight** - Minimal performance impact on VS Code

## 🔧 How It Works

Error Aavi monitors your VS Code terminal output in real-time. When it detects error keywords, it instantly plays a custom sound alert to grab your attention - perfect for long-running builds, tests, or deployments.

**Supported Error Keywords:**
- `error`
- `failed` 
- `exception`
- And more...

## 🎮 Test Your Setup

Run the test command to ensure your audio alerts are working:

1. Open Command Palette (`Ctrl+Shift+P` / `Cmd+Shift+P`)
2. Search for `Test Error Sound`
3. Execute the command
4. You should hear the alert sound
