# Change Log

All notable changes to the "error-aavi" extension will be documented in this file.

## [1.0.0] - 2024-03-04

### Added
- Initial release
- Terminal output monitoring functionality
- Error keyword detection (error, failed, exception)
- Custom MP3 sound playback on error detection
- Cross-platform support (Windows, macOS, Linux)
- Test command to manually trigger sound
- Automatic activation on VS Code startup
- Prevention of multiple simultaneous sound playbacks

### Features
- Real-time terminal output monitoring
- Case-insensitive keyword matching
- Platform-specific audio playback methods
- User-friendly error messages
- Comprehensive documentation

### Technical Details
- Built with TypeScript
- Uses VS Code Extension API
- Native audio playback (no external dependencies)
- Minimal performance impact
